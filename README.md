Projeto de Reconhecimento Facial com DeepFace

Reconhecimento Facial em Tempo Real (DeepFace)
Este projeto implementa um sistema de reconhecimento 
facial em tempo real utilizando a câmera do notebook. 
Ele compara as faces detectadas no stream de vídeo 
com um banco de dados de imagens conhecidas, 
gerando uma mensagem de reconhecimento 
(PESSOAL RECONHECIDA) junto com o nome da pessoa.

Antes de executar o projeto, recomento você deve 
ter as seguintes ferramentas instaladas:

Python version: 3.13
IDE Utilizada: (Pycharm)

Bibliotecas utilizadas
cv2
os
deepface 


Salvar suas fotos na pasta faces_db ex.: maria_silva.jpg
